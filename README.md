# Nextjs Base
这是一个基于 Next.js v15 的基础项目模板创建的练习项目。

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## integrate shadcn/ui
这个分支主要练习 shadcn/ui 的使用。
首先是集成 shadcn:
```
pnpm dlx shadcn@latest init
```
安装 button 组件测试一下：
```
pnpm dlx shadcn@latest add button
```
